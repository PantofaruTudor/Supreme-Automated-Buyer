# Supreme Automated Buyer

Acesta este un bot de automatizare creat pentru a monitoriza și achiziționa produse de pe supreme.com. Proiectul a fost dezvoltat pentru a rezolva problema stocurilor care se epuizează rapid (drop-uri).

### Ce face mai exact:
* **Monitorizare în timp real:** Verifică disponibilitatea produselor folosind BeautifulSoup.
* **Automatizare Browser:** Folosește Selenium pentru a simula pașii de cumpărare (adăugare în coș, completare date).
* **Eficiență:** Este optimizat să acționeze mult mai rapid decât un utilizator uman.

### Tehnologii folosite:
* **Python**
* **Selenium** (pentru controlul browserului)
* **BeautifulSoup4** (pentru parsarea datelor HTML)
* **Requests** (pentru interacțiuni rapide cu API-uri)
